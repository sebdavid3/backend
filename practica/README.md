# Práctica de Backend

Este repositorio contiene ejercicios de JavaScript y una práctica de backend con TypeScript + Express.

## Estructura

- `javascript/`: ejercicios básicos (arreglos, objetos, POO, JSON, etc.).
- `Typescript/`: API backend en Node.js con TypeScript.

## Requisitos

- Node.js 18+
- npm

## Ejecutar la API TypeScript

1. Entra a la carpeta del proyecto TypeScript:
   ```bash
   cd Typescript
   ```
2. Instala dependencias:
   ```bash
   npm install
   ```
3. Inicia en modo desarrollo:
   ```bash
   npm run dev
   ```

## Notas

- El servidor está implementado con Express.
- El código principal está en `Typescript/src/`.
